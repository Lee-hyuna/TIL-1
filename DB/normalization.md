## BCNF 정규형

| 학번(K)       |
| ----------- |
| **과목번호(K)** |
| 교수          |

1, 2, 3 정규형 조건은 만족하지만.

교수가 과목번호를 결정할 수 있다면

**'교수' 속성은 후보키가 아니지만 결정자이다**



#### BCNF 정규형 완성
